# Proyecto Losinno — Material Educativo

## Sobre el proyecto
Aplicación web para generar PDFs de material educativo personalizado
para pacientes de la Dra. Silvia Losinno, médica especialista en
nutrición y bienestar integral.

## Stack técnico
- HTML/CSS/JS puro (un solo archivo)
- jsPDF para generación de PDFs
- Sin frameworks ni dependencias externas

## Identidad visual
- Tipografía principal: Playfair Display
- Colores: ver assets/paleta.md
- Logo: ver assets/logo.svg

## Estructura del proyecto
- MaterialEducativo_Losinno.html → archivo principal
- assets/ → recursos visuales
- config/ → acceso Claude
- CHANGELOG.md → historial de versiones

## Instrucciones para Claude
- Siempre trabajar sobre la última versión del HTML
- Mantener el archivo como un solo .html autocontenido
- Al terminar cada sesión, hacer push de la versión actualizada
- Registrar cada cambio en CHANGELOG.md con fecha
