# Paleta de Colores Corporativos — Silvia Losinno

## Colores principales

| Nombre        | HEX       | Uso principal                        |
|---------------|-----------|--------------------------------------|
| Crema         | `#F8F6F2` | Fondo principal, fondos de página    |
| Dorado        | `#C9A66B` | Acentos, detalles decorativos        |
| Celeste       | `#A2D0D0` | Fondos secundarios, highlights suaves|
| Rosa fucsia   | `#B4475F` | Acentos vibrantes, llamadas a acción |
| Verde teal    | `#1E6E6E` | Color principal de marca, títulos    |
| Carbón        | `#3D3D3D` | Texto principal, tipografía          |

> ⚠️ Nota: El PDF muestra `#74F75` para el rosa — valor corregido a `#B4475F` según visualización real del manual.

## Uso recomendado

- **Fondo de documentos/PDFs:** `#F8F6F2` (crema)
- **Títulos y encabezados:** `#1E6E6E` (teal) o `#3D3D3D` (carbón)
- **Cuerpo de texto:** `#3D3D3D`
- **Acentos y decoración:** `#C9A66B` (dorado) y `#A2D0D0` (celeste)
- **Énfasis / destacados:** `#B4475F` (rosa)

## Variables CSS

```css
:root {
  --color-crema:   #F8F6F2;
  --color-dorado:  #C9A66B;
  --color-celeste: #A2D0D0;
  --color-rosa:    #B4475F;
  --color-teal:    #1E6E6E;
  --color-carbon:  #3D3D3D;
}
```
