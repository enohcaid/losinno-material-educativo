# Tipografía — Silvia Losinno

## Tipografías principales

### Playfair Display Bold
- **Uso:** Títulos, encabezados, nombre de marca, textos destacados
- **Peso:** Bold (700)
- **Google Fonts:** https://fonts.google.com/specimen/Playfair+Display
- **Archivo local:** `/assets/fuentes/` (incluido en fuentes.zip)

### Lato
- **Uso:** Cuerpo de texto, subtítulos, descripciones, UI general
- **Pesos recomendados:** Regular (400), Light (300), Bold (700)
- **Google Fonts:** https://fonts.google.com/specimen/Lato
- **Archivo local:** `/assets/fuentes/` (incluido en fuentes.zip)

## Jerarquía tipográfica

| Elemento        | Fuente              | Peso  | Tamaño sugerido |
|-----------------|---------------------|-------|-----------------|
| Título principal| Playfair Display    | Bold  | 32–48px         |
| Subtítulo       | Playfair Display    | Bold  | 20–28px         |
| Cuerpo de texto | Lato                | 400   | 14–16px         |
| Texto secundario| Lato                | 300   | 12–13px         |
| Etiquetas / UI  | Lato                | 700   | 10–12px         |

## Import CSS (Google Fonts)

```css
@import url('https://fonts.googleapis.com/css2?family=Playfair+Display:wght@700&family=Lato:wght@300;400;700&display=swap');
```
